## ğŸš¨ Troubleshooting

### **Service Won't Start**
```bash
# Check if port is in use
lsof -i :8080
lsof -i :8081
lsof -i :8082

# Kill existing process
kill -9 <PID>
```

### **GitHub Sync Failing**
```bash
# Verify token
echo $GITHUB_TOKEN

# Test GitHub API access
curl -H "Authorization: token $GITHUB_TOKEN" \
  https://api.github.com/user
```

### **Dashboard 404**
```bash
# Verify files exist
ls -la founder_infinity/dashboards/

# Check webhook listener logs
tail -f founder_webhook_listener.log
```

### **Reset Everything**
```bash
# Stop all services
killall python3

# Remove state files
rm -f pause.flag banned_ips.json security_audit.json

# Restart
./start_infinity_suite.sh
```
