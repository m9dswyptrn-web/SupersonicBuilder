SonicBuilder Supersonic Changelog

Added

  • Integrated **QR code** generator using `segno` for automatic labeling of build versions
    • Added **SBOM generator** to record Python environment and installed packages
      • Added **PDF metadata stamping** with profile, vehicle, and version info
        • Introduced **multi-profile matrix build** (dark / light / all)
          • Added **GitHub release publisher** with automated attachment of zips and notes
            • Included **dark/light theme toggle** and print mode in diff renderer
              • Added **Supersonic Edition tools**: diff renderer, demo PDF generators, build automation
                • Created **lean installer** for standalone Supersonic environments

                Changed

                • Reworked build pipeline for faster ZIP creation and manifest generation
                  • Simplified config file layout for easier customization
                    • Updated default color scheme for HTML diff output (cyan + amber accents)
                      • Improved cross-platform compatibility for Replit and local builds

                      Fixed

                      • Resolved file permission issues on Windows
                        • Fixed changelog parser edge cases
                          • Corrected SHA256 checksum formatting
                            • Improved QR PNG scaling and path handling
